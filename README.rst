==================
appveyor_artifacts
==================

Download artifacts from AppVeyor builds of the same commit/pull request.

* Python 2.6, 2.7, PyPy, PyPy3, 3.3, and 3.4 supported on Linux and OS X.

Quickstart
==========

Install and run:

.. code:: bash

    pip install appveyor_artifacts
    appveyor_artifacts --help
